1002(W4)中秋

何謂虛擬主機

間單的說，虛擬主機就是藉由實體主機劃分數十或是數百個獨立空間，這些空間中都有獨立的網域、網頁、資料庫、郵件伺服器、虛擬主機控制平台，這些空間就是虛擬主機。

 

虛擬伺服器，一般稱為虛擬主機 (Shared Web Hosting)，是用來放置的網頁的空間，是設計來讓瀏覽者讀取網站的地方，這是一種單個主機可以支援多個網站與服務的網路技術，虛擬主機的”虛擬”解釋不是沒有或虛空的意思，他的意思是單個實體伺服器或是伺服器群延生出更多個虛擬伺服器來提供需要的HTTP，FTP，EMAIL等多項服務。

 

由於一般企業要自行架設單一實體主機與伺服器，架設費用包含高階電腦、充足網路頻寬、MIS人員維護等等，架設成本著實不低，所以才有專業主機房業者提供虛擬主機的專業服務，提供一般企業的架站服務。

使用虛擬主機的好處

虛擬主機設計的用意主要是來降低企業自行架設主機的成本，技術上是採用實體主機以網際網路共用方式，將實體主機劃分為多個虛擬主機以提供更多企業使用，當然依照使用等幾劃分也會區分為不同規格，如頻寬限制、容量限制、功能多寡、掛載網域數量等等..

解決方式-

Git push常見問題

Q1.

remote: Invalid username or password.

fatal: Authentication failed for 'https://github.com/xxxxxxxx.git/'

A1.

這就是典型的打錯帳號密碼的案例，

帳號必須打上"倉儲名稱"，而密碼則是"Github帳戶的密碼"。

或者是太急著想要push，就會出現"Authentication failed for"的問題

這個時候代表沒有進行

git config --global user.name "xxx"

git config --global user.email "xxx"

輸入"倉儲名稱"和"Github帳戶的密碼"即可。

Q2.

git push因編輯上去檔案出現問題而無法推送

A2.

暴力推送 git push -f

如果出現問題就使用老師上課教過的指令還原。

解救英文

來源:mde.tw

曾經錯過沒有關係, 但若因為終身菜英文而錯失美好未來, 著實可惜. 解救菜英文就從現在開始.

先從最常用的 2000 個英文字彙開始.

國中英語字彙 2000 字

普通高中 4500 -7000 字

技職與綜合高中 4000 - 5500 字

自行寫程式兼學英文字彙:

假如能夠配合 https://github.com/mdecourse/lookupdict 從 Webster 英英字典中整理出 5500 字 的英文說明, 並逐字研讀, 應該是解救菜英文最佳手段.

使用電腦學習英文字彙:

另一項學習英文字彙的方式則是利用電腦或筆電下載安裝 Anki, 然後導入 4000 英文單字學習模組, 就可以開始學習了. Anki 還有其他學習英文的 Decks.

使用手機學習英文字彙:

在手機上安裝查查單字 Apps (Android 查查英文).

在手機上安裝 Anki英辞 Apps, 這是以 https://github.com/skywind3000/ECDICT 英漢字典內容為基礎的 Anki 學習套件, 使用者可以將所查詢的英文單字解釋內容設為 Anki 字詞, 進一步作為暗記英文字彙的有效工具.

Develop發展(重要)

https://github.com/mdecourse/cmsimde 的開發, 可以在一個目錄中放入 cmsimde, 然後將 up_dir 中的內容放到與 cmsimde 目錄同位階的地方, 使用 command 進入 cmsimde 目錄, 執行 python wsgi.py, 就可以啟動, 以瀏覽器 https://localhost:9443 就可以連接, 以 admin 作為管理者密碼, 就可以登入維護內容.

cmsimde 的開發採用 Leo Editor, 開啟 cmsimde 目錄中的 cmsimde.leo 就可以進行程式修改, 結束後, 若要保留網際內容, 只要將 cmsimde 外部的內容倒回 up_dir 目錄中即可後續對 cmsimde 遠端倉儲進行改版.

init.py 位於  up_dir 目錄, 可以設定 site_title 與 uwsgi 等變數.
